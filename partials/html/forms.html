Forms

Forms is builded on the same .grid .col system adding .form class. It can be applied to whole form or any .grid inside itself. (.form class affecting on all inner.col blocks)
.colclass can be applied directly on inputs, selects and textareas.


Use .colwrappers for inputs, selects and textareas if you need to make inline forms
