Grids

We use inline-blocks for columns.
In that way we preventing common problem of all float grids: variable columns height.
For example, Bootstrap trying to fix it it in flexbox-way
But this solution shrinks all colunns into a single row. So if we have more than 12 columns in .row- we have unsolved problem.
Also flexboxhave a lot of issues on IE 10 & 11.!!!
Also inline-blocks grid can be used for non-standart grids such as 7-columns with any margins:

Column gaps
Use .xs0 to hide 1th column
Use .s-6 to show 1th column and make it 6x wide
Use .s-gap-1 to make 4th column 1x gap, and .l-gap-0 to clear it.

#1
.col .xs0.s-6 .m0 .l-3
#2
.col .xs-5 .s-4 .m-7 .l-3
#3
.col .xs-3 .s-2 .m-2 .l-4
#4
.col .xs-4 .s-11 .s-gap-1 .m-2 .l-2 .l-gap-0
